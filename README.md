A simple web application built using **HTML, CSS, and JavaScript** that measures how many times a user can click within **10 seconds**. At the end of the timer, the application displays the total number of clicks and the average clicks per second (CPS).

- Features
1) 10-second countdown timer
2) Real-time click counting
3) Displays total clicks
4) Calculates average clicks per second (CPS)
5) Restart the game and play again
6) Simple and responsive user interface

- Tech Stack
1) HTML5
2) CSS
3) JavaScript

- How to Use
1. Open the application in your browser.
2. Click the **Start** button.
3. Click as fast as you can for **10 seconds**.
4. After the timer ends, your results will be displayed:
   - Total Clicks
   - Average Clicks Per Second (CPS)
5. Press **Restart** to try again.
